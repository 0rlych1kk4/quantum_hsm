Quantum HSM

A quantum-safe Hardware Security Module using Dilithium (NIST PQC standard).

Try It

curl -X POST http://127.0.0.1:8080/generate \
  -H 'Content-Type: application/json' \
  -d '{ "wallet_id": "wallet1" }'

curl -X POST http://127.0.0.1:8080/sign \
  -H 'Content-Type: application/json' \
  -d '{ "wallet_id": "wallet1", "message": "48656c6c6f", "is_hex": true }'

Why Post-Quantum?

After NIST's standardization of quantum-safe algorithms like Dilithium, using traditional ECDSA or RSA is no longer secure against future quantum attacks.
